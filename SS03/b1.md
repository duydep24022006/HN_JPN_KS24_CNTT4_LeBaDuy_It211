2.     Hoàn thành bảng so sánh sau giữa SOAP và REST (ít nhất 4 tiêu chí):
| Tiêu chí | SOAP | REST |
|---|---|---|
| Giao thức truyền tải | HTTP, SMTP, FTP | Chủ yếu HTTP |
| Định dạng dữ liệu chính | XML | JSON (có thể XML, text) |
| Trạng thái (stateful/stateless) | Có thể stateful | Stateless |
| Tính dễ dàng triển khai | Phức tạp hơn | Đơn giản hơn |

3.     Cho đoạn dữ liệu sau đang được viết dưới dạng JSON, hãy chuyển đổi sang XML tương ứng:
<sinhvien>
    <maso>SV001</maso>
    <hoten>Nguyen Van A</hoten>

    <diem>8.5</diem>
    <diem>7.0</diem>
    <diem>9.0</diem>
</sinhvien>
4.     Nêu ưu điểm của JSON so với XML trong truyền tải dữ liệu qua Web Service (ít nhất 2 điểm).
JSON có cú pháp ngắn gọn hơn XML nên dữ liệu truyền tải nhỏ hơn.
JSON dễ đọc và dễ viết hơn XML.
JSON parse nhanh hơn giúp tăng hiệu suất hệ thống.
JSON phù hợp với ứng dụng web/mobile hiện đại.
JSON được hỗ trợ rất tốt trong JavaScript và REST API.
JSON giảm băng thông mạng do payload nhỏ hơn XML.
